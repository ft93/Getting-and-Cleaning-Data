Getting-and-Cleaning-Data
=========================
This is the repository course project from coursera data science course "Getting and Cleaning Data"

Course Project

1. The source for this project is from (https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip). 
2. create file name called "run_analysis.R" inside UCI HAR Dataset folder.
3. set the working directory to UCI HAR Dataset folder.
4. use read.table() function to load the data from UCI HAR Dataset folder and bind the related data using the rbind() function
